## Описание программы

GitHub User Finder — это графическое приложение (GUI) для поиска пользователей GitHub через официальное GitHub REST API. Программа позволяет находить информацию о любом пользователе GitHub, просматривать его профиль (имя, био, количество репозиториев, подписчиков, дату регистрации и т.д.), добавлять пользователей в избранное, а также сохранять список избранных в файл JSON.

### Основные возможности:
- Поиск пользователей по username
- Добавление пользователей в избранное
- Сохранение и загрузка избранных из JSON
- Детальный просмотр профиля
- Удаление из списка избранных
- Валидация ввода данных
- Автоматическое тестирование
